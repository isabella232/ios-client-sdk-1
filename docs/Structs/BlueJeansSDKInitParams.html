BlueJeansSDKInitParams

public struct BlueJeansSDKInitParams

To be used as a parameter of the BlueJeansSDK.initialize method. Can be used to change the configurable behaviour and UX of the SDK. Note: the call to initialize must be made before any services of the BlueJeansSDK are accessed.

  • init(contentShareParameters:is5x5GalleryLayoutEnabled:playIVRs:)

    Undocumented

    Declaration

    Swift

    public init(contentShareParameters: ContentShareParametersProtocol? = nil, is5x5GalleryLayoutEnabled: Bool? = nil, playIVRs: Bool? = nil)
  • contentShareParameters

    Configures the behaviour when content is shared from the SDK. Can be set to nil if content/screen sharing is not implemented.

    Declaration

    Swift

    public var contentShareParameters: ContentShareParametersProtocol?
  • is5x5GalleryLayoutEnabled

    If set to true, then in .gallery videoLayout, there will be a maximum of 5x5 (landscape) or 8x3 (portrait) videos in a grid. Otherwise there will be a maximum of 3x3 (landscape) or 4x2 (portrait) videos. Reccomended to be set to true only on newer devices, and those with larger screens.

    Declaration

    Swift

    public var is5x5GalleryLayoutEnabled: Bool?
  • playIVRs

    If set to true, then when the meeting is recorded, an audio message indicating that recording is in progress will be played to the user. “Recording has started” “Recording has stopped” or “Recording is on”. The messages are localized into German, French, Spanish and English.

    Declaration

    Swift

    public var playIVRs: Bool?
