PermissionServiceProtocol

public protocol PermissionServiceProtocol

The permission service can be used to check and request and set the required permissions from the user to join a meeting.

  • hasRequiredPermissions

    An observable boolean, specifies whether the user has granted the required permissions to join a meeting.

    Declaration

    Swift

    var hasRequiredPermissions: BJNObservable<Bool> { get }
  • requiredPermissions

    An array of the Permission which are required to have an PermissionStatus of .authorized before we can join a meeting

    Declaration

    Swift

    var requiredPermissions: BJNObservable<[Permission]> { get }
  • setRequiredPermissions(_:)

    Sets requiredPermissions

    Declaration

    Swift

    func setRequiredPermissions(_ permissions: [Permission])
  • currentPermissions

    The current state of permissions in the SDK.

    Declaration

    Swift

    var currentPermissions: BJNObservable<[Permission : PermissionStatus]> { get }
  • requestPermission(for:onCompletion:)

    Request permissions for the Permission required to join the meeting.

    Declaration

    Swift

    func requestPermission(for permission: Permission, onCompletion: ((PermissionStatus) -> Void)?)
